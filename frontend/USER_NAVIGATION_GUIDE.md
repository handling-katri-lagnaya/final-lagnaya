# User Navigation Guide - Khatri Lagnaya

## 🎯 Navigation Based on User Role

The application now shows different navigation menus based on whether you're logged in as a **regular user** or an **admin**.

---

## 👤 Regular User Navigation

When logged in as a regular user, you'll see:

### **Desktop Navigation Bar (Top)**

- **Dashboard** - Your personal dashboard with matches and timeline
- **Matches** - Browse all your match suggestions
- **Favourites** - View profiles you've saved as favorites
- **Pricing** - View and purchase service packages
- **My Profile** - View and edit your profile
- **Settings** - Manage account settings

### **Mobile Menu (Hamburger)**

All the above options plus:

- **My Matches** - Quick access to matches
- **Logout** - Sign out of your account

### **User Profile Badge**

- Shows your name and email
- Green dot indicates you're online
- Click to access profile

---

## 🛠 Admin Navigation

When logged in as an admin, you'll see:

### **Desktop Navigation Bar (Top)**

- **Admin Dashboard** - Full admin control panel
- **User View** - Switch to regular user view

### **Mobile Menu (Hamburger)**

- **Admin Dashboard** - Access admin features
- **User View** - See the app as a regular user
- **Logout** - Sign out

### **Admin Badge**

- Shows "Admin" badge next to your name
- Indicates elevated privileges

## 📱 Available Pages by Role

### **Public Pages (No Login Required)**

- `/` - Home page
- `/process` - How it works
- `/pricing` - Pricing information (public view)
- `/contact` - Contact information
- `/login` - Login page
- `/profile/create` - Create new profile

### **User Pages (Login Required)**

- `/dashboard` - User dashboard
- `/matches` - Browse matches
- `/match/:id` - View match details
- `/favourites` - Saved profiles
- `/profile` - Your profile
- `/settings` - Account settings
- `/pricing` - Pricing packages (authenticated view)
- `/payment` - Payment processing
- `/payment-success` - Payment confirmation

### **Admin Pages (Admin Login Required)**

- `/admin` - Admin dashboard with all features:
  - Overview & Statistics
  - User Management
  - Guna Matching Tool
  - Profile Verification
  - Match Suggestion System
  - Payment Management
  - Family Meetup Requests
  - Feedback Management
  - Activity Logs

## 🚀 Quick Navigation Tips

### **For Regular Users:**

1. **Start at Dashboard** - See your matches and timeline
2. **Browse Matches** - Find compatible profiles
3. **Save Favourites** - Mark profiles you like
4. **Check Pricing** - Upgrade for more matches
5. **Update Profile** - Keep information current

### **For Admins:**

1. **Admin Dashboard** - Central control panel
2. **Verify Profiles** - Review pending profiles
3. **Generate Matches** - Create match suggestions
4. **Monitor Activity** - Track all actions
5. **Switch to User View** - Test user experience

---

## 🔄 Switching Between Views

### **Admin to User View**

1. Click "User View" in navigation
2. See the app as a regular user
3. Test features and functionality

### **User to Admin View**

1. Click "Admin Dashboard" in navigation
2. Access all admin features
3. Manage platform operations

## 🆘 Navigation Issues?

### **Can't see navigation?**

- Make sure you're logged in
- Check your user role
- Refresh the page

### **Wrong navigation showing?**

- Verify you're using correct credentials
- Clear browser cache
- Log out and log back in

### **Missing menu items?**

- Some items are role-specific
- Check if you're on the right account
- Contact support if issues persist

---

**Note**: Navigation automatically adapts based on your login status and user role. All features are accessible through the appropriate navigation menu for your account type.
